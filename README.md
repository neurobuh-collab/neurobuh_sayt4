# НейроБухгалтерия — сайт-визитка

Чистые HTML / CSS / JS, без сборки. Кидаешь папку на любой статический хостинг, и всё работает.

## Что внутри

```
index.html      разметка всех секций + мета-теги, Open Graph, микроразметка Schema.org
styles.css      токены (цвета, шрифты, тени) и вся вёрстка, включая адаптив
script.js       мобильное меню, отправка формы, кот Копейка
assets/         фото, favicon, калькулятор УСН
.nojekyll       нужен GitHub Pages, чтобы он не трогал файлы
```

## Как выложить на GitHub Pages

1. Создай репозиторий, залей туда содержимое этой папки (index.html должен лежать в корне).
2. Settings → Pages → Source: Deploy from a branch → main → /(root) → Save.
3. Через пару минут сайт откроется по адресу `https://<логин>.github.io/<репозиторий>/`.

## Что нужно доделать своими руками

**1. Форма обратной связи.** Сейчас в `index.html` стоит заглушка:

```html
<form ... action="https://formspree.io/f/ВАШ_ID" ...>
```

Пока туда не вписан реальный адрес, форма никого не отправляет, а показывает вежливое «напишите в Telegram». Чтобы заработала:

- зарегистрируйся на formspree.io, создай форму, скопируй её endpoint (вида `https://formspree.io/f/xyzabcde`);
- вставь его в `action` вместо заглушки.

Бесплатного тарифа хватает на 50 заявок в месяц. Альтернатива без сторонних сервисов — Telegram Bot API: заявка падает прямо тебе в бота. Скажи, если захочешь такой вариант, соберу.

**2. Домен и canonical.** В `<head>` стоит `<link rel="canonical" href="https://neurobuh.ru/">` и `og:image` с относительным путём. Когда будет финальный адрес сайта, поправь canonical и пропиши полный URL картинки для превью в мессенджерах.

**3. Фото.** Лежит в `assets/portrait.webp` (900×1035, 80 КБ) и `assets/portrait.jpg` — запасной вариант для старых браузеров. Меняешь оба файла, ничего в коде трогать не надо.

## Копейка

Живёт в `script.js`, весь SVG рисуется кодом, картинок нет. Состояние (сытость, ласка, дата последнего визита) хранится в localStorage браузера посетителя. С 23:00 до 7:00 по времени гостя кот спит и ворчит, если его будят.

Реплики лежат в массивах `PURR` и `LINES` в начале блока — добавляй свои, ничего больше менять не нужно.
